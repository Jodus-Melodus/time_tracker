# Time Tracker

> Track your work automatically, without getting in your way

An activity tracker designed for the workplace, yet flexible enough to use anywhere.

## üöÄ Getting started

### Installation

1. Download the latest release [here](https://github.com/Jodus-Melodus/time_tracker/releases).
2. Select `time_tracker-x86_64-pc-windows-msvc.zip` to download.
3. Move the zip file to your prefered location.
4. Right-click the  file and select `Extract`.
5. Double-click `time_tracker.exe` to launch the app.
